# Appendix E — Demo Run Artifacts

This appendix indexes the run files referenced by the main text.

## E.1. Baseline

- `runs/scenario-01/claude-opus-4.6.md` — English translation of the raw Claude Opus 4.6 baseline response to the naive prompt without an output contract.
- `prompts/scenario-01-naive.md` — the scenario text and prompt used for the baseline run.

## E.2. Structured

- `runs/scenario-02/gpt-5.4-thinking.md` — English translation of the raw GPT-5.4 structured response with an output contract.
- `prompts/scenario-02-standard.md` — the scenario text and prompt used for the structured run.

## E.3. Top-level

- `runs/scenario-02-top-level/gpt-5.4-thinking.md` — English translation of the raw GPT-5.4 top-level response with the same output contract and an additional abstraction constraint.
- `prompts/scenario-02-top-level.md` — the scenario text and prompt used for the top-level run.
